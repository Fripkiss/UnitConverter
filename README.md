# UnitConverter
Этот проект представляет собой конвертер единиц измерения с намеренно созданными проблемами "легаси-кода". Код рабочий, но содержит типичные антипаттерны и нарушения принципов чистого кода.

Основные возможности конвертации:
Расстояния:
- Километры ↔️ Мили (km ↔️ miles)
- Метры ↔️ Футы (meters ↔️ feet)
Площади:
- Гектары ↔️ Акры (hectares ↔️ acres)
  
Коэффициенты конвертации
- Километры → Мили: × 0.621371
- Мили → Километры: × 1.60934
- Метры → Футы: × 3.28084
- Футы → Метры: × 0.3048
- Гектары → Акры: × 2.47105
- Акры → Гектары: × 0.404686
